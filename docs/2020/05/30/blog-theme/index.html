Blog Theme/博客主题

2020-05-30

本博客的主题基于默认主题hugo-lithium 进行了一些微小的改动。

修改菜单栏，添加目录标签

直接在config.toml上修改，

[[menu.main]]
    name = "About"
    url = "/about/"
    weight = "2"
[[menu.main]]
    name = "Categories"
    url = "/Categories/"
    weight = "3"
[[menu.main]]
    name = "Tags"
    url = "/Tags/"
    weight = "4"
[[menu.main]]
    name = "GitHub"
    url = "https://github.com/huanghaicheng1024"
    weight = "5"

weight表示顺序。

更换图标

下载喜欢的图标的两个文件(.ico,.png),分别为博客左上角图片与浏览器标签页上面的图标，分别放在主题文件夹/static/images,/static，修改config.toml上favicon与[params.logo]的值。

添加自定义CSS

config.toml添加代码

  # Custom CSS. Put here your custom CSS files. They are loaded after the theme CSS;
  # they have to be referred from static root. Example
    [params.customCSS]
     href = "css/my-style.css"

并在主题static文件夹建立对应文件，添加自己所需的CSS样式，比如我取消了图标的弧度以及使R绘图的caption居中

.nav-logo img {
    max-height: 50px;
    border-radius: 0%;
  }
  
  
@media (min-width: 600px) {
  .nav-logo img {
    max-height: 50px;
    border-radius: 0%;
  }
  
.caption {
  text-align: center;
}
