本博客的主题基于默认主题hugo-lithium 进行了一些微小的改动。传送门

在使用过程中有什么不满意的地方会自己尝试修改，并在这里不定期更新。

修改菜单栏，添加目录标签

直接在config.toml上修改，

[[menu.main]]
    name = "About"
    url = "/about/"
    weight = "2"
[[menu.main]]
    name = "Categories"
    url = "/Categories/"
    weight = "3"
[[menu.main]]
    name = "Tags"
    url = "/Tags/"
    weight = "4"
[[menu.main]]
    name = "GitHub"
    url = "https://github.com/huanghaicheng1024"
    weight = "5"

weight表示顺序。

2020-6-1

反正没什么人会点击我的GitHub，改写了一下footer.html，可以自定义类似RSS feed的footer内容。只需在config.toml设置

[[menu.footer]]
    name = "GitHub"
    url = "https://github.com/huanghaicheng1024"
    weight = "1"

更换图标

下载喜欢的图标的两个文件(.ico,.png),分别为博客左上角图片与浏览器标签页上面的图标，分别放在主题文件夹/static/images,/static，修改config.toml上favicon与[params.logo]的值。

添加自定义CSS

config.toml添加代码

  # Custom CSS. Put here your custom CSS files. They are loaded after the theme CSS;
  # they have to be referred from static root. Example
    [params.customCSS]
     href = "css/my-style.css"

并在主题static文件夹建立对应文件，添加自己所需的CSS样式，比如我取消了图标的弧度以及使R绘图的caption居中以及最大宽度为100%，以防手机页面体验不好；添加了支持中文字体的微软雅黑。

body {
  font-family: 'Merriweather','Microsoft Yahei', serif;
}

h1, h2, h3, h4, h5, h6, .nav, .article-duration, .archive-item-link, .footer {
  font-family: 'Lato','Microsoft Yahei', sans-serif;
}

.caption {
  text-align: center;
}

.figure img{
  max-width: 100%;
}


.nav-logo img {
    max-height: 50px;
    border-radius: 0%;
  }
  
  
@media (min-width: 600px) {
  .nav-logo img {
    max-height: 50px;
    border-radius: 0%;
  }
